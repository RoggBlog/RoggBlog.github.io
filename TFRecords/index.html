TFRecords 文件的读取

解析

解析 TFRecords 的 example 协议内存块

  • serialized 标量字符串 Tensor，一个序列化的 Example
  • features 字典数据，键为读取的名字，值为 FixedLenFeature
  • shape 输入数据的形状，一般不指定，为空列表
  • dtype 输入数据类型，与储存进文件的类型要一致
